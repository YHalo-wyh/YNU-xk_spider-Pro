# YNU选课助手 Pro

> Forked from [starwingChen/YNU-xk_spider](https://github.com/starwingChen/YNU-xk_spider)

云南大学选课辅助工具，支持课程监控、自动抢课、微信推送等功能。

## ✨ 功能特性

- 🔐 自动登录（验证码自动识别，全程无需手动操作）
- 📚 课程浏览（支持多板块、搜索、翻页）
- 🎯 多课程同时监控
- 📱 微信推送通知（余课提醒 + 抢课成功）
- 🔄 浏览器异常自动恢复
- 🎨 现代化界面

### 使用说明

1. **登录**
   - 输入学号和密码，选择选课轮次
   - 点击「登录」按钮，程序会自动打开浏览器
   - 登录过程全自动，验证码自动识别，无需任何操作
   - 浏览器窗口仅供查看登录进度

2. **选课监控**
   - 登录成功后，在程序界面中浏览课程
   - 使用搜索框快速定位目标课程
   - 点击课程卡片上的「加入待抢」按钮
   - 可添加多门课程到待抢列表

3. **开始抢课**
   - 确认待抢列表中的课程
   - 点击「开始监控」按钮
   - 程序会自动轮询监控所有待抢课程，发现余量立即抢课
   - 支持多门课程同时监控，按顺序轮流检测
   - 抢课成功会弹窗提示并自动从列表移除

> ⚠️ **注意**：请尽量在程序界面中操作，避免直接操作浏览器窗口

## 📱 微信推送配置

使用 [Server酱](https://sct.ftqq.com/) 实现微信推送：

1. 访问 https://sct.ftqq.com/
2. 微信扫码登录，获取 SendKey
3. 在程序中展开「📱 微信推送」设置
4. 勾选启用并填入 SendKey

推送内容：
- **余课提醒**：监控到课程有空位时推送（每门课仅提醒一次）
- **抢课成功**：成功选上课程时推送
