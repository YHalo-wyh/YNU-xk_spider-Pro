# YNU选课助手 Pro v2.2.0

## 🎯 功能扩展版本：新增跨专业课程（FAWKC）支持

本次版本聚焦“课程类型扩展”和“发布流程一致性”：在现有查询/监控/选课链路中新增跨专业课程模块，并同步升级版本与发布产物命名。

---

## 🚀 核心更新

### 1. 新增「跨专业课程」模块
- 课程类型新增：`跨专业课程`。
- 内部类型映射新增：`cross_major -> FAWKC`。
- UI 下拉框可直接选择跨专业课程并发起查询。

### 2. 查询链路支持 FAWKC
- 课程查询 `teachingClassType` 现支持 `FAWKC`。
- 跨专业课程查询接口走 `elective/programCourse.do`。
- 保持与主修/体育课程一致的查询参数结构。

### 3. 选课与结果核实链路保持一致
- 抢课提交继续使用 `elective/volunteer.do`（`addParam` 结构），支持 `teachingClassType=FAWKC`。
- 选课结果核实继续使用 `elective/courseResult.do`。
- 与现有监控线程、自动重登、状态核实机制无缝兼容。

### 4. 版本与构建同步升级
- 程序版本升级至 `v2.2.0`。
- 打包产物命名同步升级至 `v2.2.0`。
- 保持主程序与 `Watchdog.exe` 一起发布。

---

## 🛠️ 兼容性说明

- 本次新增能力基于教务系统返回的课程类型码 `FAWKC`。
- 若学校端后续调整跨专业课程类型码，请在 `xk_spider/gui/config.py` 中同步映射。

## 🧭 升级建议

1. 关闭旧版本程序（若在监控中，请先点击“停止”）。
2. 使用 `v2.2.0` 安装包升级，或解压便携版到新目录运行。
3. 登录后选择「跨专业课程」，先执行一次课程查询验证接口可用性，再开始监控。

---

## ⚠️ 注意事项

- 由于教务系统接口策略可能调整，跨专业课程在不同学院/学期的开放范围可能不同。
- 若出现“查询成功但无课程”，请先确认当前学期是否开放跨专业选课。
